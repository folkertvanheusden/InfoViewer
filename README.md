infoviewer
----------

View all kinds of information.
Sources can be mqtt, output from a program (one-shot or tail) and static texts.
Data from sources can be evaluated if it is json.


build
-----

packages required:
* libsdl1.2-dev
* libsdl-gfx1.2-dev
* libsdl-image1.2-dev
* libsdl-ttf2.0-dev
* libmosquitto-dev
* libconfig++-dev
* libjansson-dev

The example uses fonts from fonts-freefont-ttf
